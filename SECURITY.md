# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 7.x     | :white_check_mark: |
| < 7.0   | :x:                |

## Reporting a Vulnerability

To report a security vulnerability, please send an email to [security@d3js.org](mailto:security@d3js.org) (or the appropriate maintainer email if different).

We will verify the vulnerability and, if confirmed, release a patch as soon as possible.
